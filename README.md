# HRMS Hub
URL-https://hrms-hub-8d5b.vercel.app/

HRMS Hub is a Human Resource Management System built using React, TypeScript, and Vite. It helps manage employees, attendance, and dashboards efficiently.

## 🚀 Features
- Employee Management
- Attendance Tracking
- Dashboard Analytics
- Modern UI using Tailwind CSS

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## 📦 Installation

```bash
git clone https://github.com/saurabhkumar-creator/hrms-hub.git
cd hrms-hub
npm install
npm run dev
